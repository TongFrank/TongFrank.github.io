查看 documentation 获得有关hexo的更多信息，以下为Hexo的一些简单命令操作。

快速开始

初始化并安装所需组件：

$ hexo init      # 初始化
$ npm install    # 安装组件

创建博客文章

$ hexo new "My New Post"  #创建博客文章

更多信息请参考: Writing

启动Hexo服务

$ hexo server/hexo s  #启动Hexo服务/启动预览

更多信息请参考: Server

生成静态文件

$ hexo generate/hexo g  #生成页面

更多信息请参考: Generating

部署到远程站点

$ hexo deploy/hexo d  #部署到服务器
