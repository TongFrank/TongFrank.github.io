本博客网站基于 Hexo构建，使用的主题为ParticleX，非常感谢大佬们提供的主题，本篇文章为Hexo自动生成的，现在加以修改，以纪念本博客上线。

查看 documentation 获得有关hexo的更多信息，以下为Hexo的一些简单命令操作。

Quick Start

初始化并安装所需组件：

hexo init      # 初始化
npm install    # 安装组件

Create a new post

$ hexo new "My New Post" #创建博客文章

More info: Writing

Run server

$ hexo server/hexo s #启动Hexo服务/启动预览

More info: Server

Generate static files

$ hexo generate/hexo g #生成页面

More info: Generating

Deploy to remote sites

$ hexo deploy/hexo d #部署到服务器
